	-다단은 레이아웃을 여러개의 컬럼으로 쪼개서 구성한다는 의미
	
flex(플렉서블 레이아웃)
	대부분 웹사이트는 수직 구성
	레이아웃 구성할 때 가장 많이 사용하는 요소들이 대부분 블록으로 
	설정 되어 있음
	수직으로 쌓이기 때문에 수직 구성은 쉽지만 수평 구성이 어렵다.
	float / inline-block / table

flex-wrap
	플렉스 라인에 더 이상 여유가 없을 때, 플렉스 요소의 위치를
	다음 줄로 넘길지 여부를 설정
	nowrap --> 기본값, 넘어가지 않음, 요소의 너비자체를 줄여서 한줄에 배치
	wrap --> 여유 공간이 없으면 다음 줄로 넘김
	wrap-reverse --> 여유 공간이 없으면 (위쪽)다음줄로 넘김

display속성에 block이나 inline, inline-block은 자기 자신의 표현 방식을 결정 짓지만
flex 속성은 자기 하위 요소 (자식)의 표현 방식을 결정 짓는다.

flex 컨테이너(display:flex;)로 설정되어있는 요소(이 요소의 자식은 flex영향 받음)
flex 항목: 부모가 flex 속성으로 설정되어있는 자식들, 실질적으로 영향받는다.
주축(main axis): 플렉스 컨테이너 안에서 플렉스 항목을 배치하는 방향
		    기본은 왼쪽에서 오른쪽, 수평방향
